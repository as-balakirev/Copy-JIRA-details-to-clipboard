The addon is used to copy values of all specified labels from issue details section in JIRA.

A jira label wrapped in { } returns it's value. A "{ticketNo}" and "{ticketTitle}" are tied for JIRA number and JIRA summary respectively. Template string consists of JIRA labels and any text.

For example, template: "{ticketNo} / {ticketTitle} / {Type} / {Priority}", copies to the clipboard a string: "EXMPL-123 / Example JIRA summary / Story / Medium".
