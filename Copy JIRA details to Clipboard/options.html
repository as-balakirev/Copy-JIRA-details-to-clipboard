The addon adds "Copy to Clipboard" button to the jira ticket menu. Button is used to copy values from details section in jira to the clipboard for all specified labels.

A jira label wrapped in { } returns it's value. Template string can consist of JIRA labels and any text.

Additional Values:

  • {ticketNo} - returns a short jira ticket. For example - EXMPL-123;
  • {ticketTitle} - returns a jira summary;
  • {ticketUrl} - returns a ticket url. Supports rich text. Returns a plain url (e.g. https://jira/browse/EXMPL-123) OR html url - EXMPL-123, if supported by application.

For example, template: "{ticketNo} / {ticketTitle} / {Type} / {Priority}", copies to the clipboard a string: "EXMPL-123 / Example JIRA summary / Story / Medium".
